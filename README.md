### 1. Prompt - "Create a 3D game using HTML, CSS and JavaScript"

#### Output (Code Provided)

Here's **Abyss Drift**, a 3D deep-sea dodge game in a single HTML file (Three.js for rendering, plain CSS and JS for everything else).

**How to Play:**
You're a bioluminescent drifter sinking into the deep. Steer with arrow keys or WASD (touch and drag on mobile) to weave between rock pillars, bobbing jellyfish, and spinning urchins.

Speed ramps up the deeper you go, and your depth in meters is your score, with a best-run tracker.

#### Output (Code Provided)

**ABYSS MAZE** - a 3D first-person maze game for the terminal.

Pure C++17, zero external dependencies.

```bash
// Build:
// Linux/macOS : g++ -O2 -std=c++17 abyss_maze.cpp -o abyss_maze
// Windows     : g++ -O2 -std=c++17 abyss_maze.cpp -o abyss_maze.exe
//               (or cl /O2 /std:c++17 abyss_maze.cpp)
```

**Goal:** Collect all 5 relics (*) then find the glowing exit (E).

**Controls:**

* W / S : Move forward / back
* A / D : Turn left / right
* Q / E : Strafe left / right
* M     : Toggle minimap
* X     : Quit
